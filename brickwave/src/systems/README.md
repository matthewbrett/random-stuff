# Systems

This directory contains game systems and managers.

## Structure

- **PhaseSystem.js** - Manages phase brick timing and synchronization
- **ScoreManager.js** - Handles scoring, coins, and echo charges
- **InputManager.js** - Input handling and control remapping
- **SaveManager.js** - localStorage persistence for saves and settings
- **LevelLoader.js** - Loads and parses Tiled JSON maps
- **AudioManager.js** - Manages music and sound effects
- **ParticleManager.js** - Particle effect system
- **CameraController.js** - Camera following and smoothing

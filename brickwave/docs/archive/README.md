# Archived Documentation

## Completed Plans

### ✅ Health & Sprite System (Completed 2025-12-06)
**[HEALTH_SPRITE_PLAN.md](HEALTH_SPRITE_PLAN.md)**
- Difficulty-based health system (Easy: 5♥, Medium: 4♥, Hard: 3♥)
- Animated player sprite (cat character with armor)
- Game over screen with retry/quit
- Invincibility frames (1.5s)
- All 7 phases fully implemented

### ✅ Resolution System (Completed 2025-12-06)
**[RESOLUTION_PLAN.md](RESOLUTION_PLAN.md)**
- SCALE multiplier system (1x for Retro, 2x for Polished)
- Retro mode: 320×180 with 8×8 tiles
- Polished mode: 640×360 with 16×16 tiles
- All entities, UI, and systems scaled
- No level file changes required

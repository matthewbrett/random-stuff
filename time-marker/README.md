# Time Marker

A simple, elegant web app that displays a live clock and lets you mark moments in time.

## Features

- **Live Clock Display** - Shows current time in HH:MM:SS format
- **Mark Moments** - Capture the current time with a single button click
- **Elapsed Time Tracking** - See how much time has passed since your most recent mark
- **Optional Descriptions** - Add text notes to any marked moment
- **Individual Deletion** - Remove marks you no longer need
- **Mobile Responsive** - Works beautifully on all screen sizes

## Tech Stack

- **Pure Vanilla JavaScript** - No frameworks or dependencies
- **Single HTML File** - Everything (HTML, CSS, JS) in one portable file
- **Modern CSS** - Gradient background, glassmorphism effects, smooth animations
